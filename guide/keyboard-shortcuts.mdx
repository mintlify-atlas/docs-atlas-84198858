Izzy supports full keyboard navigation to control the app without using your mouse.

## Global shortcuts

These shortcuts work system-wide, even when Izzy is not the active window.

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Option + Space` | Show/hide Izzy | Customizable in Settings |
| Media keys | Control playback | Uses macOS native media key integration |

<Tip>
  You can customize the global hotkey modifier in Settings. Choose from Option, Control, Command, or Shift + Option.
</Tip>

<Warning>
  `Command + Space` is the default Spotlight shortcut on macOS. If you choose Command as your modifier, you may need to change Spotlight's shortcuts in **System Settings** > **Keyboard** > **Keyboard Shortcuts**.
</Warning>

## Application shortcuts

These shortcuts work when Izzy is the active window.

### Playback control

| Shortcut | Action |
|----------|--------|
| `Space` | Play/pause current song |
| `→` | Next track |
| `←` | Previous track |
| `↑` | Volume up |
| `↓` | Volume down |

### Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd + F` | Focus search bar |
| `Cmd + ,` | Open settings |
| `Cmd + Q` | Quit Izzy |
| `Tab` | Navigate between UI elements |
| `Enter` | Activate selected item |
| `Esc` | Close modals or clear focus |

## Tips for efficient navigation

<AccordionGroup>
  <Accordion title="Use keyboard-only navigation">
    Press `Tab` to move between buttons, search fields, and other interactive elements. Press `Enter` to activate the focused element.
  </Accordion>
  
  <Accordion title="Quick search access">
    Press `Cmd + F` from anywhere in the app to immediately focus the search bar and start typing.
  </Accordion>
  
  <Accordion title="Toggle visibility quickly">
    Use the global hotkey (`Option + Space` by default) to show and hide Izzy instantly without minimizing or closing the app.
  </Accordion>
  
  <Accordion title="Control volume precisely">
    Hold `↑` or `↓` to smoothly adjust volume. Release when you reach your desired level.
  </Accordion>
  
  <Accordion title="Media key integration">
    If your keyboard has media keys (play/pause, next, previous), they'll work with Izzy even when other apps are active.
  </Accordion>
</AccordionGroup>

## Customizing shortcuts

Currently, only the global hotkey modifier can be customized:

<Steps>
  <Step title="Open settings">
    Click the gear icon (⚙️) or press `Cmd + ,`.
  </Step>
  
  <Step title="Find the global hotkey section">
    Scroll to the **Global Hotkey** card.
  </Step>
  
  <Step title="Choose your modifier">
    Select from:
    - **Option** (⌥): Default, recommended
    - **Control** (⌃): Alternative modifier
    - **Command** (⌘): May conflict with Spotlight
    - **Shift + Option** (⇧⌥): Two-key combination
  </Step>
  
  <Step title="Test your new shortcut">
    The shortcut updates immediately. Press your new combination + Space to verify it works.
  </Step>
</Steps>

<Note>
  If a hotkey registration fails (usually due to conflicts with other apps), Izzy will automatically revert to the previous working shortcut and show an error message in Settings.
</Note>

## Accessibility

All keyboard shortcuts follow macOS accessibility guidelines:

- **VoiceOver compatible**: All controls can be accessed with VoiceOver enabled
- **Focus indicators**: Clear visual feedback shows which element has keyboard focus
- **Logical tab order**: Tab navigation follows a natural reading order
- **Escape hatches**: Press `Esc` to exit modals and cancel operations

## Modifier key reference

For users new to macOS, here's a reference of the modifier keys:

| Symbol | Key name | Location on keyboard |
|--------|----------|----------------------|
| ⌘ | Command | Next to Space bar |
| ⌥ | Option (Alt) | Next to Command |
| ⌃ | Control | Far left, bottom row |
| ⇧ | Shift | Far left and right, second row |
| ␣ | Space | Center, bottom row |
